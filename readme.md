# imam

- imam 是一个用来记住一些不常用的账户密码网站
- 我无法保障该项目的安全，所以请不要放置一些重要的密码



- 项目使用的是 spring boot + vue.js 的前后端分离方案
- 没有使用mybatis  而是 mybatis -plus
- ui 使用的是 vant  仅适配移动端
